# PixlArt — Пиксельный редактор

**Командный учебный проект**

Участники:  
- Борзенко Владислав  
- Байбара Максим (h0xy13377)

## ✨ Возможности

- **Полноэкранная обложка** с плавной анимацией
- **Интерактивная сетка** пикселей (16×16 / 32×32 / 48×48)
- **Палитра** из 9 цветов
- **Инструменты**:
  - Карандаш (рисование с зажатой кнопкой)
  - Ластик
  - **Заливка** с красивой staggering-анимацией (anime.js)
- **Автоматическое сохранение** в LocalStorage (рисунок и настройки сохраняются)
- **Скачивание** готовой картинки в PNG
- Адаптивный дизайн (работает на компьютере и планшете)

## 🚀 Демо

[Открыть сайт](https://pixel-art-editor-five.vercel.app)

## 🛠 Технологии

- HTML5
- CSS3 (Grid + Flexbox)
- Vanilla JavaScript
- anime.js (анимации)
- dom-to-image (экспорт в PNG)
