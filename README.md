# PixlArt — Пиксельный редактор

**Командный учебный проект**

Участники:  
- Борзенко Владислав  
- Байбара Максим

## Возможности

- Рисование карандашом
- Ластик
- Заливка с анимацией
- Палитра цветов
- Изменение размера холста
- Скачивание изображения
- Адаптивная вёрстка

## Как запустить

Открой файл `index.html` в браузере.

## Технологии

HTML, CSS, JavaScript, anime.js, dom-to-image
